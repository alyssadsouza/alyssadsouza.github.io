Program courses

In my program I've had the opportunity to learn about the following concepts so far:

  • CS 137: Programming Principles
    • Topics include procedures and parameter passing, arrays and structures, recursion, sorting, pointers and simple dynamic structures, space and time analysis of designs, and design methodologies. The course is taught using the C programming language.
  • SE 101: Introduction to Methods of Software Engineering
    • Course objectives include:
      1. describing key software engineering activities, including requirements elicitation, design, and testing;
      2. writing code to control a small computer (e.g., Arduino or Raspberry Pi);
      3. using a source code version control system
